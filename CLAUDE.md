This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Overview

This is an HTML and CSS practice repository containing learning exercises organized into two main folders:
- `A-Html-Folder-Practice/`: HTML fundamentals and examples
- `B-Css-Folder-Practice/`: CSS styling techniques and examples

The repository is bilingual (English/Spanish) and serves as a learning resource for web development basics.

## Repository Structure

### HTML Practice Files (`A-Html-Folder-Practice/`)
HTML files are organized with alphanumeric prefixes indicating topic progression:
- **A-series**: HTML structure and meta tags (A1-Estructura.html, A2-MetaTags.html, etc.)
- **B-series**: Multimedia elements (images, video, audio)
- **C-series**: Lists
- **D-series**: Tables
- **E-series**: Grouping elements (div, section, etc.)
- **F-series**: Iframes
- **G-series**: Forms (including backend integration)
- **H-series**: SVG

### CSS Practice Files (`B-Css-Folder-Practice/`)
CSS examples follow a similar structure:
- HTML files demonstrate CSS concepts
- Corresponding stylesheets are in the `styles/` subdirectory
- **A-series**: CSS application methods (external, inline, page-level)
- **B-series**: Selectors (element, ID, class, combinators, descendant)
- **C-series**: Box model (margin, borders, padding, width/height, display, position, overflow/float)
- **D-series**: Colors and backgrounds
- **E-series**: Text and fonts
- **F-series**: Icons
- **G-series**: Lists styling
- **H-series**: Tables styling
- **I-series**: Pseudo-classes

## Code Style

- All HTML files include extensive Spanish comments explaining each element
- Files are educational in nature with clear examples of each concept
- CSS files are linked from HTML files using relative paths: `href="styles/[filename].css"`
- Standard HTML5 structure with `<!DOCTYPE html>`, `<head>`, and `<body>` sections
